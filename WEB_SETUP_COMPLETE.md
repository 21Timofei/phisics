## 🚀 Как запустить

### Вариант 1: Быстрый старт (3 минуты)

```bash
# 1. Активируйте виртуальное окружение (если есть)
source venv/bin/activate  # macOS/Linux
# venv\Scripts\activate   # Windows

# 2. Установите/обновите зависимости
pip install -r requirements.txt

# 3. Запустите приложение
streamlit run web/streamlit_app.py
```

Приложение откроется в браузере на `http://localhost:8501`

## 📖 Как использовать

### Первый эксперимент

1. **Откройте приложение** (автоматически откроется браузер)

2. **Настройте в боковой панели:**
   - Число кубитов: `1`
   - Тип канала: `Depolarizing`
   - Параметр p: `0.1` (используйте слайдер)
   - Shots: `1000`
   - Метод: `LSQ`

3. **Нажмите:** 🚀 Запустить томографию

4. **Изучите результаты:**
   - Посмотрите на Process Fidelity (должна быть > 0.95)
   - Изучите матрицу Чой
   - Посмотрите на операторы Крауса
   - Изучите 3D визуализацию на сфере Блоха

### Примеры экспериментов

#### Эксперимент 1: Влияние параметра шума

```
Кубиты: 1
Канал: Depolarizing
p = 0.1  → Запустить → Записать Fidelity
p = 0.2  → Запустить → Записать Fidelity
p = 0.3  → Запустить → Записать Fidelity

Наблюдайте снижение fidelity с ростом шума!
```

#### Эксперимент 2: Сравнение методов

```
Кубиты: 1
Канал: Amplitude Damping
γ = 0.3
Shots: 1000

Метод: LSQ → Запустить → Записать время и fidelity
Метод: MLE → Запустить → Записать время и fidelity

MLE точнее, но медленнее!
```

#### Эксперимент 3: Статистический анализ

```
Кубиты: 1
Канал: Phase Damping
λ = 0.2
Shots: 1000
Множественные прогоны: ✓
Число прогонов: 20

Получите статистику и стандартное отклонение!
```

#### Эксперимент 4: 2-кубитная система

```
Кубиты: 2
Канал: Two-Qubit Depolarizing
p = 0.1
Shots: 3000
Метод: LSQ

Изучите более сложную систему!
```

## 🌐 Деплой в интернет

### Самый простой способ: Streamlit Cloud (Бесплатно!)

1. **Загрузите код на GitHub:**
   ```bash
   git add .
   git commit -m "Add web interface"
   git push origin main
   ```

2. **Зайдите на:** https://share.streamlit.io/

3. **Нажмите "New app"** и заполните:
   - Repository: `your-username/phisics`
   - Branch: `main`
   - Main file: `web/streamlit_app.py`

4. **Нажмите "Deploy!"**

5. **Получите публичный URL!** Например:
   `https://your-username-phisics-streamlit-app.streamlit.app`

См. подробную инструкцию в **[DEPLOY.md](DEPLOY.md)**:

## 📊 Характеристики

### Поддерживаемые каналы

**1-кубитные:**

- Phase Damping Channel
- Random CPTP Channel

**2-кубитные:**
- Two-Qubit Depolarizing
- Random CPTP Channel

### Производительность

| Конфигурация | Время выполнения |
|--------------|------------------|
| 1 кубит, 1K shots, LSQ | ~5 сек |
| 1 кубит, 5K shots, MLE | ~30 сек |

## 🔧 Решение проблем

### Приложение не запускается

```bash
# Проверьте установку Streamlit
streamlit --version

# Переустановите зависимости
pip install -r requirements.txt --force-reinstall

# Проверьте версию Python
python --version  # Должна быть 3.11+
```

### Ошибка импорта noiselab

```bash
# Убедитесь, что вы в корневой директории проекта
pwd  # Должно быть /Users/timofei21/phisics

# Экспортируйте PYTHONPATH
export PYTHONPATH="${PYTHONPATH}:$(pwd)"

# Или запускайте из корня:
streamlit run web/streamlit_app.py
```

### Медленная работа

1. Уменьшите shots до 500
2. Используйте LSQ вместо MLE
3. Отключите шум измерений
4. Используйте 1 кубит вместо 2

### Браузер не открывается автоматически

```bash
# Запустите с явным указанием порта
streamlit run web/streamlit_app.py --server.port 8501

# Затем откройте вручную
# http://localhost:8501
```
