# 📂 Структура проекта NoiseLab++

│   │   │
│   │   ├── tomography/             🔬 Томография

│   │   │   ├── reconstruction.py   • Реконструкция
│   │   │   └── state_prep.py       • Подготовка состояний
│   │   │
│   │   ├── representations/        🔄 Представления
│   │   │   ├── choi.py             • Матрица Чой
│   │   │   ├── ptm.py              • PTM
│   │   │   └── kraus_decomp.py     • Декомпозиция Крауса
│   │   │
│   │   └── metrics/                📏 Метрики
│   │       ├── fidelity.py         • Process fidelity
│   │       ├── distance.py         • Расстояния
│   │       └── validation.py       • Валидация
│
├── 🧪 ТЕСТЫ
│   └── tests/
│       ├── test_core.py            • Тесты базовых компонентов
│       ├── test_channels.py        • Тесты каналов
│       └── test_tomography.py      • Тесты томографии
│
├── 📝 ПРИМЕРЫ
│   └── examples/
│       └── demo_tomography.py      • Демонстрация Python API
│
└── 🔧 КОНФИГУРАЦИЯ
    ├── requirements.txt             📦 Зависимости Python
    ├── .gitignore                   🚫 Игнорируемые файлы
    └── .streamlit/
        └── config.toml              ⚙️ Настройки Streamlit

## Ключевые файлы

### 🚀 Для запуска

| Файл | Описание | Команда |
|------|----------|---------|
| `web/streamlit_app.py` | Веб-интерфейс | `streamlit run web/streamlit_app.py` |
| `web/app.py` | Flask API | `python web/app.py` |
| `examples/demo_tomography.py` | Демо | `python examples/demo_tomography.py` |

### 📚 Для изучения

| Файл | Содержание |
|------|------------|
| `WEB_SETUP_COMPLETE.md` | ⭐ **НАЧНИТЕ ЗДЕСЬ!** Полная инструкция |
| `QUICKSTART.md` | Быстрый старт за 3 минуты |
| `WEB_INTERFACE.md` | Описание всех возможностей интерфейса |
| `README.md` | Основная документация проекта |

### 🔧 Для настройки

| Файл | Назначение |
|------|-----------|
| `requirements.txt` | Зависимости Python |
| `.streamlit/config.toml` | Тема и настройки интерфейса |
| `.gitignore` | Игнорируемые файлы Git |

## Потоки данных

### Веб-интерфейс → Backend → Результаты

```
┌─────────────────┐
│  Пользователь   │
│   (Streamlit)   │
└────────┬────────┘
         │
         │ 1. Выбирает параметры
         │    (кубиты, канал, shots)
         ↓
┌─────────────────┐
│  create_channel │
│  (streamlit_app)│
└────────┬────────┘
         │
         │ 2. Создает канал
         ↓
┌─────────────────┐
│  QPT Algorithm  │
│  (qpt.py)       │
└────────┬────────┘
         │
         │ 3. Выполняет томографию
         │    • Подготовка состояний
         │    • Применение канала
         │    • Измерения
         │    • Реконструкция
         ↓
┌─────────────────┐
│  Visualization  │
│  (Plotly)       │
└────────┬────────┘
         │
         │ 4. Отображает результаты
         ↓
┌─────────────────┐
│  Пользователь   │
│  (Результаты)   │
└─────────────────┘
```
